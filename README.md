# 手势控制桌面工具

基于OpenCV实现的手势识别和桌面控制功能，通过简单的手势即可控制计算机的各种操作。

## 功能特点

- 实时手势识别
- 丰富的手势控制功能
- 美观的用户界面
- 交互性强，用户体验好

## 手势对照表

| 手势 | 功能 |
|------|------|
| 张开手掌 | 显示手势状态 |
| 食指上指 | 鼠标移动并单击 |
| 胜利手势 (V) | 双击 |
| 竖大拇指向上 | 向上滚动 |
| 竖大拇指向下 | 向下滚动 |
| OK手势 | 右键单击 |
| 握拳 | 暂停/恢复控制 |

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 使用说明

1. 运行程序后，点击"开始控制"按钮
2. 将手放在摄像头前，确保手部清晰可见
3. 做出相应的手势来控制计算机
4. 按空格键或使用握拳手势可暂停/恢复控制
5. 按'q'键退出程序

## 系统要求

- Python 3.6+
- 摄像头
- Windows/Mac/Linux

## 技术栈

- OpenCV - 图像处理和手部轮廓检测

## 工作原理

本项目使用计算机视觉技术检测手部轮廓和手指数量：
1. 通过摄像头捕获实时视频流
2. 使用高斯模糊和二值化处理图像
3. 检测手部轮廓和凸包缺陷
4. 根据手指数量和位置识别手势
5. 将识别到的手势映射到相应的桌面控制操作

## 注意事项

- 确保在光线充足的环境下使用，以获得更好的识别效果
- 使用时请保持手部在摄像头视野范围内
- 如需更精确的手势识别，可以考虑下载MediaPipe模型文件

## 扩展功能

本项目设计具有良好的扩展性，可以轻松添加新的手势识别功能：
- 添加新的手势类型到get_gesture函数
- 定义对应的操作到execute_gesture_command函数
- 在用户界面中更新手势说明
